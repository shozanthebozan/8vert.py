## 8vert.py instructions
**What it is:**
8vert.py is a unit-converting (eg. mm to cm) program made with python. It is still not completed yet.

**How to use it:**
1. Open up the program in a supported code runner like IDLE or vs code (don't use single-line code runners like python in cmd)
2. If prompted to install an extension to run python, or anything python-related, install it

3. Run the code
4. It will display: `Enter 1 for distance/length/height, 2 for weight, 3 for speed, 4 for temp, 5 for volume, 6 for area, 7 for time or 8 for data:`, enter the corresponding number to what type of converting you want to do (eg. 1 for distance/length/height)
5. After you enter a number, it will display: `Enter 1st unit in lowercase, it can be mm,cm,m,km,inches,feet,yards or miles:`, this text depends on what you chose before this. Enter the corresponding number. For my example, ill choose mm.
6. Then it'll display: `Enter 2nd unit in lowercase, it can be mm,cm,m,km,inches,feet,yards or miles:`, enter the second unit. If i entered mm in the first one, and km in this, the conversion will be mm to km.

7. After that it'll display: `Enter mm amount:`, it'll say something else instead of mm if you didn't select it

8. Lastly it'll display: `That is 0.00106 km.`, the amount reported depends on what you selected as the second unit, and the amount you entered, and what you selected for first unit. This has demonstrated a conversion from 1060 mm to km.
